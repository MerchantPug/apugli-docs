# Ignore Calming Aura

[Entity Type Tag](../tags.md).

Defines which entity types should be ignored by the `toomanyorigins:calming_aura` power.

Location: `toomanyorigins/tags/entity_types/ignore_calming_aura`

## Default Values
- `minecraft:fox`
- `minecraft:llama`
- `minecraft:hoglin`
- `minecraft:wolf`

# Swing Hand

[Entity Action](../entity_actions.md).

An entity action that swings the hand of a player entity.

Type ID: `apugli:swing_hand`

### Fields

*None.*

### Example
```json
"entity_action": {
    "type": "apugli:swing_hand"
}
```
This action swings the hand of a player entity.

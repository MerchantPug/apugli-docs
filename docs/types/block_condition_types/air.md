# Air

[Block Condition Type](../block_condition_types.md).

Checks whether the block is a type of air (`minecraft:air`, `minecraft:cave_air`, etc). 

Type ID: `apugli:air`

### Fields

*None.*


### Example
```json
{
    "type": "apugli:air"
}
```

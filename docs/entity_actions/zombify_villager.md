# Zombify Villager

[Entity Condition](../entity_conditions.md).

Turns a villager entity into a zombie villager.

Type ID: `apugli:zombify_villager`

### Fields

*None.*

### Example
```json
"entity_action": {
    "type": "apugli:zombify_villager"
}
```
This action zombifies a villager.

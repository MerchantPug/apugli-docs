Defines which blocks a player with the [Dragonborn](../../origins/toomanyorigins_rework/dragonborn.md) origin has.

Location: `toomanyorigins/tags/blocks/lightning_rods`

## Default Values
- `minecraft:furnace`
- `minecraft:smoker`
- `minecraft:blast_furnace`
